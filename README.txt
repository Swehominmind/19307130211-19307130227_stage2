CZ_CPU_ver2.5 for Windows

0. codes文件夹是源代码，CZ_CPUver2.5文件夹中是exe程序和运行所需要的库，以及测试样例（放在test文件夹中）
1. Windows64位系统进入文件夹CZ_CPUver2.5, 运行Demo.exe, 所需运行环境已经配置好, 直接运行即可. 
2. 在上方的输入栏中输入要测试的样例，按enter确认加载该样例
3. 加载后按左方run或next1即可测试（next1运行一步，run会持续运行，可拖动速度条设置运行速度）
4. 按reset以重置，可以再次运行测试.
5. 可以按debug按钮进入调试界面, 可以观察cpu运行的阶段, 并可添加断点(先在右方选择, 是要在哪个阶段添加断点, FDEMW或是delete, 您也可以选择在断点上点击删除断点)
6. 可以测试并通过y86-code和y86-code-honor中所有测试数据，以及自己写的iopq测试数据（123.yo）
7. 可以将其他yo文件放入 \CZ_CPUver2.5\test中，运行测试
